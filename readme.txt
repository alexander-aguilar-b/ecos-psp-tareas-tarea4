INSTRUCCIONES PARA DESCARGAR EL PROYECTO DESDE GITHUB Y EJECUTARLO LOCALMENTE
*********************************************************************************************************************************************************		  
1-Acceder a la siguiente URL del repositorio en github: https://github.com/alexander-aguilar-b/ecos-psp-tareas-tarea4
2-Descargar el ZIP del proyecto (ecos-psp-tareas-tarea4-master.zip)
3-Descomprimir el archivo anterior.

5-Ejecutar el comando: mvn package
